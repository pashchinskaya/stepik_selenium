# Selenium Webdriver: Автоматизация перехода по вкладкам и решение CAPTCHA

Это демонстрационный скрипт на **Python**, использующий библиотеку **Selenium Webdriver** для выполнения следующих задач:
1. Открытие веб-страницы.
2. Клик по кнопке, вызывающей перенаправление на новую вкладку/окно.
3. **Переключение фокуса** с исходной вкладки на вновь открытую.
4. Считывание числового значения (**`x`**) со страницы.
5. Выполнение математического вычисления (CAPTCHA) по заданной формуле: $result = \log(|12 \cdot \sin(x)|)$.
6. Ввод полученного результата в текстовое поле.
7. Отправка формы.
8. Обработка **`alert`**-окна для извлечения финального ответа/кода.

---

## Как запустить

### Предварительные требования

Для запуска скрипта вам понадобится:

* **Python 3.x**
* Установленная библиотека **Selenium**:
    ```bash
    pip install selenium
    ```
* **Веб-драйвер** (например, ChromeDriver) для вашего браузера, который должен быть доступен в `PATH` вашей системы.

### Запуск скрипта

1.  Сохраните код как файл, например, `tab_solver.py`.
2.  Запустите его из командной строки:
    ```bash
    python tab_solver.py
    ```

> Скрипт автоматически откроет браузер (Chrome), выполнит все шаги и выведет финальный ответ в консоль.

## Основные используемые методы Selenium

В этом скрипте демонстрируются ключевые возможности автоматизации, связанные с работой окон/вкладок:

| Метод | Описание |
| :--- | :--- |
| `current_window_handle` | Получение идентификатора (хэндла) текущей активной вкладки. |
| `window_handles` | Получение списка всех открытых идентификаторов вкладок/окон. |
| `switch_to.window(handle)` | Переключение фокуса Webdriver на конкретную вкладку. |
| `switch_to.alert` | Переключение на модальное окно (`alert`, `confirm`). |
| `alert.text` | Получение текста из модального окна. |
| `alert.accept()` | Закрытие модального окна (аналог нажатия "ОК"). |
| `browser.quit()` | Корректное закрытие браузера и завершение сессии. |
